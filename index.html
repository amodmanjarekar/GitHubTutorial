Set up

Check required installations:

  • git -
    • - run git --version
    - expect git version 2.34.1.windows.1 or similar
    - otherwise to install, check this.
  • GitHub CLI
    • - run gh --version
    - expect gh version 2.4.0 (2021-12-21) or similar
    - otherwise to install, check this.
  • node.js and npm
    • - run node --version
    - expect v16.14.0 or similar
    - run npm --version
    - expect 8.3.1 or similar
    - otherwise to install, check this.
  • npx
    • - run npx --version - expect 8.3.1 or similar
    - otherwise to install, run npm i npx

Run the project:

  • Fork a repo
  • Clone the repo
  • Launch a development server

Make Changes:

  • Pull changes from main
  • Push changes to main
  • Branching (optional, if you're interested)
